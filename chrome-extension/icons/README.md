# Extension Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16px
- `icon32.png` - 32x32px
- `icon48.png` - 48x48px
- `icon128.png` - 128x128px

Optional (for connection status indicators):
- `icon16-gray.png` - 16x16px (disconnected state)
- `icon32-gray.png` - 32x32px (disconnected state)
- `icon48-gray.png` - 48x48px (disconnected state)
- `icon128-gray.png` - 128x128px (disconnected state)
- `icon16-red.png` - 16x16px (error state)
- `icon32-red.png` - 32x32px (error state)
- `icon48-red.png` - 48x48px (error state)
- `icon128-red.png` - 128x128px (error state)

## Design Guidelines

- Use a distinctive icon that represents element selection/inspection
- Suggested concepts: crosshair, target, cursor with element, inspector tool
- Colors: Primary blue (#4A90E2) to match the UI
- Background: Transparent PNG
